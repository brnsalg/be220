# Teste Técnico Be220!

### Como subir o APP?

- Baixe ou clone este repositório
- Tenha _npm_ em seu ambiente
- Rode: _npm install_
- Rode: _npm start_

### Foi Utilizado:

- **Frontend**: Ionic/Angular
- **Backend**: Firebase

### Funcionalidades:

O aplicativo oferece os seguintes destaques:

- Login
- Criar Conta
- Logout
- Responsividade
